#### 变量声明与定义的关系

声明（declaration）使名字为程序所知。
定义（definition）负责创建与名字关联的实体。

只声明一个变量而不定义它，就在变量名前面添加关键字 extern：

```c++
extern int i;//声明 i 而并未定义 i
int j;//声明并定义 j
```

如果包含了初始值，就非声明而是定义，也就是 extern 无效了：

```C++
extern double pi = 3.14; //这是定义了 pi
```
