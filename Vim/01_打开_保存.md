# 打开，保存

#### :vim test.cpp 打开文件

~~~C++
vim test.cpp
vim test_1.cpp test_2.cpp test_3.cpp//一次打开多个文件
~~~

#### vim -o2 打开 2 个水平分割的窗口

~~~C++
vim -o2
vim -o5//打开 5 个水平分割的窗口
vim -o5 test_1.cpp test_2.cpp//打开 5 个水平分割的窗口,前两个窗口分别显示 test_1.cpp 和 test_2.cpp 的内容
~~~

#### :q 退出 vim

~~~c++
:q
:quit
~~~

#### :w 保存文件

~~~C++
:w
:write
~~~

#### :wq 保存文件并退出 vim

~~~C++
:wq
~~~

#### :q! 不保存文件并退出 vim

~~~C++
:q!
:quit!
~~~
